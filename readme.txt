=== URL Preview ===
Tags: link preview, post, excerpt, Facebook type preview, linkedin type preview
Requires at least: 3.3
Tested up to: 4.0

A plugin used to embed the preview of a link similar to facebook and linkedin

== Description ==
This plugin can be used to fetch the excerpt of an external website. The excerpt would contain the title, description and image. 

It also has an option page, which can be used to add CSS to change the look and feel.

== Screenshots ==
1. Displays the button in the wordpress toolbar
2. Clicking the button, displays this popup.
3. After entering the url and clicking ok, the preview gets copied in the editor.
